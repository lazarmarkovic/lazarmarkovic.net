# Working in this repo

## Commits & PRs

- Never add `Co-Authored-By: Claude ...` (or any AI co-author trailer) to commits or PRs.
- Commit message format:
  - Subject line: imperative mood, no period, ideally under 50 chars (e.g. `Update theme submodule to latest upstream`).
  - Blank line, then body (optional) explaining *why*, wrapped at ~72 chars.
- PR format:
  - Title: short, imperative, under 70 chars.
  - Body: brief `## Summary` (bullets) and, when relevant, `## Test plan` (checklist).
